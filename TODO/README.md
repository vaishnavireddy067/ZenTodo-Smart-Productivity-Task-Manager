# ZenTodo - Premium Task Manager

ZenTodo is a modern, minimalist todo application built with a focus on simplicity, productivity, and aesthetics. It features a glassmorphic design, smooth animations, and a responsive interface that works beautifully across all devices.

## Features

- ✨ **Premium Design**: Modern glassmorphic interface with vibrant background animations.
- 🌓 **Dark Mode**: Support for both light and dark themes with persistent user preference.
- � **Global Search**: Instantly find any task with the real-time search bar.
- 📅 **Due Dates**: Set deadlines for your tasks. Overdue tasks are highlighted in red.
- 🏷️ **Categorization**: Group tasks by Work, Personal, Shopping, Health, and more.
- �📊 **Progress Tracking**: Dynamic progress bar showing your completion percentage.
- 🚩 **Priorities**: Assign Low, Medium, or High priority to your tasks with visual badges.
- ✏️ **Inline Editing**: Simply click on a task text to edit it on the fly.
- 🎉 **Confetti Celebration**: Enjoy a burst of confetti whenever you complete a task!
- 💾 **Persistence**: Your tasks are automatically saved to your browser's local storage.
- 🔍 **Filtering**: Easily filter tasks by "All", "Active", and "Completed".
- 📱 **Responsive**: Fully optimized for mobile, tablet, and desktop screens.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom design tokens and glassmorphism.
- **JavaScript**: Pure Vanilla JS for lightweight performance.
- **Lucide Icons**: Crisp, customizable vector icons.
- **Google Fonts**: Inter for clean typography.
