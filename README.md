# 🧘 ZenTodo – Smart Productivity Task Manager

> **Organize your tasks. Boost your productivity. Stay focused.**

ZenTodo is a modern productivity-focused task management application that helps users efficiently organize their daily work, track progress, prioritize important tasks, and improve time management through a clean and intuitive interface.

---

## ✨ Features

- ✅ Create, edit, and delete tasks
- 📅 Set due dates
- 🏷️ Organize tasks with categories
- ⭐ Priority levels (High, Medium, Low)
- ✔️ Mark tasks as completed
- 🔍 Search and filter tasks
- 📊 Productivity dashboard
- 🌙 Dark & Light mode
- 📱 Responsive design
- 💾 Local storage / Database support
- 🔔 Task reminders (Optional)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/ZenTodo-Smart-Productivity-Task-Manager.git
```

### Navigate to the project

```bash
cd ZenTodo-Smart-Productivity-Task-Manager
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```
